Section 11: SEO & Metadata Injection

Metadata drives discoverability. In this section, we’ll dynamically inject <title>, meta descriptions, Open Graph tags, and JSON-LD structured data into your pages using JavaScript. You’ll see clear examples for each and ChatGPT prompts to automate snippet generation.

Tip: Keep your <title> under 60 characters and meta descriptions under 160 characters for optimal SEO results.

11.1 Dynamic Document Title

Use JavaScript to set the page title based on your content:

// JS to set the document title
const pageTitle = "Awesome Guide – Section 11";
document.title = pageTitle;

11.2 Meta Description & Viewport

Inject a meta description and viewport tag into the <head>:

// JS to inject meta tags
const description = "Learn how to inject SEO metadata dynamically for better search rankings.";
const metaDesc = document.createElement("meta");
metaDesc.name = "description";
metaDesc.content = description;
document.head.appendChild(metaDesc);

const viewport = document.createElement("meta");
viewport.name = "viewport";
viewport.content = "width=device-width, initial-scale=1.0";
document.head.appendChild(viewport);

11.3 Open Graph Tags

Boost social sharing with Open Graph meta tags:

// JS to inject Open Graph tags
const ogData = {
  title: "Awesome Guide Section",
  type: "article",
  url: window.location.href,
  image: "https://example.com/og-image.png",
  description: "A quick overview of SEO metadata injection."
};
Object.entries(ogData).forEach(([prop, content]) => {
  const meta = document.createElement("meta");
  meta.setAttribute("property", "og:" + prop);
  meta.content = content;
  document.head.appendChild(meta);
});
ChatGPT Prompt: "Provide JavaScript to create and append Open Graph meta tags for title, type, url, image, and description."

11.4 JSON-LD Structured Data

Include structured data for rich search results:

// JS to inject JSON-LD structured data
const ldJson = {
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Section 11: SEO & Metadata Injection",
  "author": { "@type": "Person", "name": "StackPilot" },
  "datePublished": "2025-05-09",
  "description": "Learn to inject structured JSON-LD data dynamically."
};
const ldScript = document.createElement("script");
ldScript.type = "application/ld+json";
ldScript.textContent = JSON.stringify(ldJson);
document.head.appendChild(ldScript);
