Section 2: Getting Started

Before you write any code, it’s important to get your tools set up. This section gives you options, guidance, and ChatGPT prompts to help tailor the workflow to your comfort level.

Choosing a Code Editor

StackPilot Standard: We use Visual Studio Code.

It’s free, powerful, and has great extensions like Live Server for testing and GitHub integration.

Other Options: Sublime Text, Atom, WebStorm

Prompt: Help me decide which code editor to use for web development.

Setting Up Git & GitHub

StackPilot Standard: GitHub + GitHub Desktop for version control and backups.

GitHub makes it easy to collaborate, save your work online, and deploy your site for free.

Other Options: Git CLI, GitKraken, Bitbucket

Prompt: Help me set up GitHub Desktop and create a new repo for my modular website.

Running a Local Server

StackPilot Standard: python3 -m http.server or npx serve to preview changes in a browser.

Browsers block JS fetch() from reading local files — so you need to serve the site with HTTP.

Other Options: Live Server (VSCode extension), Apache, XAMPP

Prompt: How can I preview my HTML site locally with a simple server?
