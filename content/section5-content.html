Section 5: Creating New Sections

StackPilot makes expanding your website modular and simple. Follow this guide to add new sections while keeping your layout clean and reusable.

Step 1: Create a Content File

Each section lives in /content as an HTML fragment. For example, Section 5’s content lives in content/section5-content.html. It should not include <html>, <head>, or <body> — just the content.

Prompt: Create a new file in /content called section6-content.html. Use semantic HTML and avoid full structure tags.

Step 2: Register the Section in Navigation

Open assets/nav.js and add a new entry to the navItems array. Each section must have a unique id and a clear label.

{ id: "section6", label: "Your Section Title Here" },

Step 3: Write for Humans and ChatGPT

Your content should teach real users while being readable by GPT. Use clean structure, short paragraphs, and clearly labeled prompts.

Prompt: Add a prompt block to help users create new HTML sections with GPT.

Tips to Keep It Modular

  • Never duplicate the full HTML structure
  • Use semantic headings and sections
  • Apply reusable classes (e.g. .stackpilot-box, .stackpilot-prompt)
  • Write like you're teaching both a beginner and GPT
