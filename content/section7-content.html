Section 7: Responsive Layouts & Breakpoints

In this section, we'll explore how to build responsive layouts using three approaches: Tailwind CSS utilities, raw CSS media queries, and Bootstrap's grid system. Each example shows dynamic injection into <main> so both human readers and ChatGPT can follow along.

Tip: Responsive design ensures your site looks great on any device.

7.1 Tailwind CSS Responsive Utilities

Tailwind provides mobile-first responsive utilities—prefix any utility with a breakpoint like sm:, md:, or lg:.

<div class="bg-gray-200 p-4 sm:bg-blue-100 md:bg-green-100 lg:bg-red-100">
  <p class="text-sm sm:text-base md:text-lg lg:text-xl">
    This box changes color and text size at different breakpoints.
  </p>
</div>
ChatGPT Prompt: "Generate Tailwind CSS classes for a responsive two-column layout that stacks on mobile and shows side-by-side on md screens."

7.2 Raw CSS Media Queries

Using plain CSS, define breakpoints with @media rules.

<style>
  .responsive-box {
    background-color: #eee;
    padding: 1rem;
  }
  @media (min-width: 640px) {
    .responsive-box { background-color: #add8e6; }
  }
  @media (min-width: 768px) {
    .responsive-box { background-color: #90ee90; }
  }
  @media (min-width: 1024px) {
    .responsive-box { background-color: #ffcccb; }
  }
</style>
<div class="responsive-box">
  <p>Box color changes at sm, md, lg.</p>
</div>
ChatGPT Prompt: "Write CSS media queries for breakpoints at 640px, 768px, and 1024px to adjust a grid layout from one to three columns."

7.3 Bootstrap Grid System

Leverage Bootstrap’s grid classes to create responsive columns.

<div class="row">
  <div class="col-12 col-md-6">Column 1</div>
  <div class="col-12 col-md-6">Column 2</div>
</div>
