Section 8: Integrating a JavaScript Framework

In this section, we'll demonstrate how to slot in a lightweight JS framework—React (with a note on Vue/Svelte)—into our modular setup. We'll inject framework-specific components into <main> without overhauling the existing navigation or styling.

Tip: You don’t need a full SPA setup—just hydrate components where you need interactivity.

8.1 Why Integrate a Framework?

  • Component-based logic for dynamic UIs
  • Reusable code chunks with state management
  • Seamless pairing: humans write markup, ChatGPT fills boilerplate

8.2 React Example

Inject a simple counter component:

<div id="react-counter-root"></div>
<script type="text/javascript" src="https://unpkg.com/react@18/umd/react.development.js"></script>
<script type="text/javascript" src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
<script>
  function Counter() {
    const [count, setCount] = React.useState(0);
    return (
      React.createElement('div', { className: 'p-4 bg-gray-100' },
        React.createElement('p', null, `Count: ${count}`),
        React.createElement('button', { onClick: () => setCount(count + 1), className: 'mt-2 px-3 py-1 bg-blue-500 text-white rounded' }, 'Increment')
      )
    );
  }
  ReactDOM.createRoot(document.getElementById('react-counter-root')).render(
    React.createElement(Counter)
  );
</script>

8.3 Vue/Svelte Notes

If you prefer Vue or Svelte, the pattern is similar:


<div id="vue-widget">{{ message }}</div>
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script>
  Vue.createApp({
    data() { return { message: 'Hello from Vue!' } }
  }).mount('#vue-widget')
</script>
